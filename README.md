# 🍽️ Django Recipe Management System

A full-stack Recipe Management System built using Django that allows users to add, view, update, search, and delete recipes with image support through a clean and responsive web interface.

## ⚙️ Installation

1. Clone Repository

```bash
git clone https://github.com/manishmanimidipally/django-recipe-project.git
```

2. Navigate to Project

```bash
cd django-recipe-project
```

3. Create Virtual Environment

```bash
python -m venv venv
```

4. Activate Environment

```bash
venv\Scripts\activate
```

5. Install Dependencies

```bash
pip install -r requirements.txt
```

6. Apply Migrations

```bash
python manage.py migrate
```

7. Run Server

```bash
python manage.py runserver
```
